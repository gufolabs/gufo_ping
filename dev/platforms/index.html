    Linux x86_64 Glibc 2.24 (Debian 9)
    Linux aarch64 Glibc 2.24 (Debian 9) 2 2 2 2 2
    Linux x86_64 Musl 1.1 3 3 3
    Linux aarch64 Musl 1.1 4 4 4 4 4
    MacOS x86_64
    MacOS arm64 5 5 5 5 5
